# 🎟️ Ticket System

Microservices app to manage events, tickets and users.

## 🧱 Tech Stack
- Node.js + Express
- MongoDB
- Docker + Docker Compose
- RabbitMQ
- React (frontend)
- JWT Auth
- Swagger for API docs

## 📦 Services
- User Service
- Event Service
- Ticket Service
- Notification Service
- API Gateway

## ▶️ Run Locally

```bash
docker-compose up --build
